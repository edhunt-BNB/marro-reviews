# Marro 5-Star Reviews

A curated collection of 5-star Trustpilot reviews for Marro, categorized by theme.

## Categories

- **High Quality Ingredients/Product** - Reviews praising our fresh, real ingredients
- **Customer Service** - Praise for our support team
- **Fussy Cats** - Picky cats who love our food
- **Took a While But Now Love It** - Cats who needed time but now adore it
- **Subscription Benefits** - Convenience and flexibility praise
- **Changed My Cat's Life / Health Benefits** - Health transformation stories

## Automation

Reviews are automatically fetched from Slack every 2 hours and categorized using AI.

## Setup (Internal)

Requires GitHub Secrets:
- `SLACK_BOT_TOKEN` - Slack bot token for Marro Trustpilot Sorter app
- `GROQ_API_KEY` - Groq API key for AI categorization
